# Nutrient Skills

This repository contains AI agent skills for [Nutrient](https://www.nutrient.io/) APIs and SDKs.

Each skill lives under `plugins/<skill-name>/skills/<skill-name>/SKILL.md`. Read the relevant SKILL.md for instructions on how to use a skill.

## Available Skills

- **nutrient-document-processor-api** — Convert, extract, transform, and secure documents via the Nutrient Document Web Services API.
